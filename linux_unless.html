不要なパッケージの削除

$ sudo apt-get --purge remove `deborphan` //不要（孤立）パッケージ
$ sudo dpkg -P `deborphan --find-config` //不要設定ファイル
$ sudo apt-get clean
$ sudo apt-get autoremove

du ディレクトリ容量を調べる

$ sudo du -sh /*  // カレントディレクトリ以下の容量
$ sudo du -sh /* | sort -hr  //容量でソート
$ sudo du -sh /hoge/hoge/* | sort -hr | head -15 //上位１５ファイル

apache2 のログ、PHPでエラー出まくると肥大化する。

$ sudo log | gerp hoehoge > log.new //hogehogeのログ書き出し
$ sudo log | gerp --invert-match hogehgoe > log.new //hogehoge以外のログ書き出し
